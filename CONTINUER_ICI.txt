ÉTAPES POUR FINALISER (après composer install):
================================================================================

1. VÉRIFIER QUE COMPOSER EST TERMINÉ
   Dans votre terminal, attendez de voir:
   "Generating optimized autoload files"
   "Discovered Package: ..."

2. GÉNÉRER LA CLÉ D'APPLICATION
   Commande:
   php artisan key:generate

3. CRÉER LA BASE DE DONNÉES
   Ouvrez MySQL et exécutez:
   CREATE DATABASE gestion_salles CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

4. CONFIGURER .ENV (si nécessaire)
   Ouvrez .env et modifiez:
   DB_DATABASE=gestion_salles
   DB_USERNAME=root
   DB_PASSWORD=votre_mot_de_passe

5. LANCER LES MIGRATIONS
   Commande:
   php artisan migrate

6. DÉMARRER LE SERVEUR LARAVEL
   Commande:
   php artisan serve

7. OUVRIR L'APPLICATION
   Dans votre navigateur:
   http://localhost:8000

================================================================================
OU UTILISEZ LE SCRIPT AUTOMATIQUE:
================================================================================

Double-cliquez sur: start.bat

Ce script fera automatiquement les étapes 2, 5 et 6.

TESTER L'APPLICATION:

1. Créer une salle:

EN CAS DE PROBLÈME:
================================================================================

❌ Erreur "Class not found":
   → Relancez: composer dump-autoload

❌ Erreur de connexion MySQL:
   → Vérifiez les identifiants dans .env
   → Vérifiez que MySQL est démarré

❌ Erreur "No application encryption key":
   → Lancez: php artisan key:generate

❌ Page blanche:
   → Vérifiez que Vite est démarré (npm run dev)
   → Vérifiez la console du navigateur (F12)
