📱 ACCÉDER À L'APPLICATION:
================================================================================

Ouvrez votre navigateur et allez sur:
👉 http://localhost:8000

Vous devriez voir:
- Un formulaire pour créer une salle
- Un tableau listant les salles (vide au début)

================================================================================
🧪 TESTER L'APPLICATION:
================================================================================

1. CRÉER UNE SALLE:
   - Remplissez le formulaire:
     * Code: A101
     * Nom: Salle de cours A101
     * Type: Cours
     * Capacité: 30
     * Bâtiment: Bâtiment A
     * Étage: 1
   - Cliquez sur "Créer la salle"
   - La salle apparaît dans le tableau

2. CRÉER PLUSIEURS SALLES:
   - Ajoutez A102, A103, B201, etc.
   - Toutes apparaissent dans le tableau

3. DÉSACTIVER UNE SALLE:
   - Cliquez sur "Désactiver" dans le tableau
   - La salle disparaît immédiatement

4. TESTER LA VALIDATION:
   - Essayez de créer une salle sans code → Erreur
   - Essayez de créer une salle avec capacité 0 → Erreur
   - Essayez de créer deux salles avec le même code → Erreur

================================================================================
🔌 TESTER L'API DIRECTEMENT:
================================================================================

Vous pouvez tester l'API avec Postman ou curl:

GET http://localhost:8000/api/salles
→ Liste toutes les salles actives

POST http://localhost:8000/api/salles
Content-Type: application/json
{
  "code": "A101",
  "nom": "Salle A101",
  "type": "Cours",
  "capacite": 30,
  "batiment": "Bâtiment A",
  "etage": 1
}
→ Crée une nouvelle salle

PUT http://localhost:8000/api/salles/1
Content-Type: application/json
{
  "code": "A101",
  "nom": "Salle A101 - Rénovée",
  "capacite": 35
}
→ Modifie la salle ID 1

DELETE http://localhost:8000/api/salles/1
→ Désactive la salle ID 1
